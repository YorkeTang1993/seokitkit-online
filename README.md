# SEOKit

Free online SEO tools — meta tag generators, schema markup builders, site checkers, and content analyzers. No signup required.

**[Try it live at seokit.site](https://seokit.site)**

[![Live Site](https://img.shields.io/badge/Live%20Site-seokit.site-blue?style=for-the-badge)](https://seokit.site)

## About

SEOKit provides free, browser-based SEO tools for webmasters, content creators, and marketers. Generate meta tags, structured data, and sitemaps. Check page speed, broken links, and SSL certificates. Analyze keyword density, readability, and heading structure. Every tool runs entirely in your browser with no server processing, no signup, and no limits.

## Tools

### Generators
- [Robots.txt Generator](https://seokit.site/robots-txt-generator) — Create custom robots.txt files with an easy visual editor
- [Meta Tag Generator](https://seokit.site/meta-tag-generator) — Create optimized title tags, meta descriptions, Open Graph and Twitter Card tags
- [Schema Markup Generator](https://seokit.site/schema-markup-generator) — Create JSON-LD structured data for articles, products, FAQs, and more
- [Open Graph Generator](https://seokit.site/open-graph-generator) — Create og:title, og:description, og:image tags for social media sharing
- [Twitter Card Generator](https://seokit.site/twitter-card-generator) — Create summary and large image card meta tags for X (Twitter)
- [XML Sitemap Generator](https://seokit.site/sitemap-generator) — Create sitemaps with priority, change frequency, and last modified date
- [.htaccess Generator](https://seokit.site/htaccess-generator) — Create redirect rules, GZIP compression, cache headers, and Apache config
- [Hreflang Tag Generator](https://seokit.site/hreflang-tag-generator) — Create hreflang tags for multilingual websites
- [Favicon Generator](https://seokit.site/favicon-generator) — Create favicons from text or emoji in all required sizes
- [QR Code Generator](https://seokit.site/qr-code-generator) — Create QR codes for URLs, text, or any data instantly
- [Lorem Ipsum Generator](https://seokit.site/lorem-ipsum-generator) — Generate placeholder text by paragraphs, sentences, or words

### Checkers
- [Redirect Chain Checker](https://seokit.site/redirect-checker) — Trace redirect paths, see 301/302 redirects and final destination
- [Keyword Density Checker](https://seokit.site/keyword-density-checker) — Analyze keyword distribution and find overused or underused terms
- [Social Share Preview](https://seokit.site/og-image-preview) — Preview how your URL looks when shared on Facebook, X, LinkedIn, and WhatsApp
- [Page Speed Checker](https://seokit.site/page-speed-checker) — Analyze page load time and simulate Core Web Vitals scores
- [Broken Link Checker](https://seokit.site/broken-link-checker) — Find dead links, 404 errors, and redirect chains
- [SSL Checker](https://seokit.site/ssl-checker) — Verify SSL/TLS certificate status, expiration, and configuration
- [Domain Age Checker](https://seokit.site/domain-age-checker) — Look up domain registration date, age, and WHOIS details
- [HTTP Header Checker](https://seokit.site/http-header-checker) — Inspect HTTP response headers, status codes, and security headers
- [Heading Structure Checker](https://seokit.site/heading-structure-checker) — Analyze heading hierarchy (H1-H6) and detect SEO heading issues

### Validators
- [XML Sitemap Validator](https://seokit.site/xml-sitemap-validator) — Check sitemap.xml for errors and validate URL structure
- [Structured Data Tester](https://seokit.site/structured-data-tester) — Validate JSON-LD and Schema.org markup for rich snippet eligibility
- [JSON-LD Validator](https://seokit.site/json-ld-validator) — Validate structured data markup, check required fields, and detect syntax errors

### Utilities
- [Canonical URL Generator](https://seokit.site/canonical-url-generator) — Create rel=canonical tags to prevent duplicate content issues
- [URL Slug Generator](https://seokit.site/slug-generator) — Convert text into SEO-friendly URL slugs
- [Google SERP Simulator](https://seokit.site/google-serp-simulator) — Preview how your page appears in Google search results
- [Anchor Text Analyzer](https://seokit.site/anchor-text-analyzer) — Analyze anchor text distribution and find over-optimized links
- [Readability Checker](https://seokit.site/readability-checker) — Calculate Flesch-Kincaid readability score and grade level
- [Word Counter](https://seokit.site/word-counter) — Count words, characters, sentences, paragraphs, and estimate reading time
- [Character Counter](https://seokit.site/character-counter) — Count characters with and without spaces, letters, digits, and punctuation
- [URL Shortener](https://seokit.site/url-shortener) — Generate short, shareable URLs from long links
- [UTM Builder](https://seokit.site/utm-builder) — Create UTM-tagged URLs for Google Analytics campaign tracking
- [Keyword Extractor](https://seokit.site/keyword-extractor) — Extract frequent keywords and phrases from text for SEO analysis
- [Text Case Converter](https://seokit.site/text-case-converter) — Convert text between UPPERCASE, lowercase, Title Case, camelCase, and more
- [HTML to Text](https://seokit.site/html-to-text) — Strip HTML tags and convert markup to clean plain text
