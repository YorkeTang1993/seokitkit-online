# DevToolKit

Free online developer tools — converters, generators, formatters, and utilities. No signup required.

**[Try it live at devtoolkit-online.vercel.app](https://devtoolkit-online.vercel.app)**

[![Live Site](https://img.shields.io/badge/Live%20Site-devtoolkit--online.vercel.app-blue?style=for-the-badge)](https://devtoolkit-online.vercel.app)

## About

DevToolKit provides free, browser-based developer tools that help you convert, generate, format, and calculate — all without leaving your browser. Your data stays on your device.

## Tools

### Converters
- [TOML to YAML](https://devtoolkit-online.vercel.app/toml-to-yaml) — Convert TOML configuration to YAML format
- [YAML to TOML](https://devtoolkit-online.vercel.app/yaml-to-toml) — Convert YAML files to TOML format
- [SVG to JSX](https://devtoolkit-online.vercel.app/svg-to-jsx) — Convert SVG markup to JSX for React components
- [SQL to MongoDB](https://devtoolkit-online.vercel.app/sql-to-mongodb) — Convert SQL queries to MongoDB queries

### Generators
- [Cron Expression Generator](https://devtoolkit-online.vercel.app/cron-expression-generator) — Build and validate cron schedules visually
- [ENV File Generator](https://devtoolkit-online.vercel.app/env-file-generator) — Generate .env files from templates
- [Nginx Config Generator](https://devtoolkit-online.vercel.app/nginx-config-generator) — Generate Nginx configuration files

### Formatters
- [JSON to Zod](https://devtoolkit-online.vercel.app/json-to-zod) — Convert JSON objects to Zod validation schemas

### Utilities
- [Chmod Calculator](https://devtoolkit-online.vercel.app/chmod-calculator) — Calculate Unix file permissions
- [HMAC Generator](https://devtoolkit-online.vercel.app/hmac-generator) — Generate HMAC signatures for API authentication

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19 + TypeScript
- **Styling:** Tailwind CSS 4
- **Hosting:** Vercel

## Privacy

All tools run 100% client-side in your browser. No data is sent to any server. No accounts, no tracking cookies, no data collection.
